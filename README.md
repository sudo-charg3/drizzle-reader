## 😤 The Problem

Reading PDFs sucks. On laptop — ugly default viewers, no customization, eye strain. On phone — pinch-to-zoom hell, no reflow, washed-out white backgrounds at 2am.

**Drizzle Reader was built to fix that.**

---

## ✨ What is Drizzle Reader?

Drizzle Reader is a **fast, free, open-source PWA PDF reader** designed for a cozy, aesthetic, premium reading experience — whether you're on your laptop, tablet, or phone. No account. No backend. No nonsense. Just you and your document.

---

## 🚀 Features

### 📖 Built for Real Reading
- Smooth, distraction-free PDF rendering
- Handles complex documents without breaking a sweat
- Instant load — no backend, no auth wall, no waiting

### ✏️ Math & LaTeX Support
Full rendering of mathematical notation and LaTeX expressions — perfect for academic papers, textbooks, research docs, and anything with equations.

### 🎨 15 Handcrafted Themes
Not just "dark mode and light mode." Drizzle ships with **15 carefully designed themes** built for aesthetic, cozy, premium reading sessions:

| Theme | Vibe |
|-------|------|
| ☀️ Parchment | Warm, paper-like, easy on the eyes |
| 🌙 Midnight | Deep dark for late-night sessions |
| 🌿 Forest | Earthy greens, nature-inspired calm |
| 🌊 Ocean | Cool blues, focused and crisp |
| 🌸 Sakura | Soft pink, gentle and serene |
| ☕ Café | Coffee tones, perfect with a warm drink |
| ...and 9 more | Each designed with intention |

### 📱 Progressive Web App (PWA)
Install it like a native app on any device. Works offline. No app store required.

### ⚡ Fast & Private
- Everything runs **in your browser** — files never leave your device
- Zero servers processing your documents
- Zero telemetry, zero tracking

## 🛠️ Tech Stack

| Layer | Technology |
|-------|-----------|
| Framework | React / Next.js |
| PDF Rendering | PDF.js |
| Math & LaTeX | KaTeX / MathJax |
| Styling | Tailwind CSS |
| PWA | next-pwa / Workbox |
| Deployment | Vercel |
| State | Zustand |
| Storage | Browser IndexedDB / LocalStorage |

> No database. No Supabase. No authentication. No cloud. Completely client-side.

## 🏃 Getting Started

```bash
# Clone the repo
git clone https://github.com/yourusername/drizzle-reader.git
cd drizzle-reader

# Install dependencies
npm install

# Start dev server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) and drop in a PDF.

---

## 📦 Build & Deploy

```bash
# Production build
npm run build

# Start production server
npm start
```

Deploys seamlessly to Vercel, Netlify, or any static host.
